#

> [!Important]
> - Requires **Google Chrome** to obtain the Token.
> - If the Cover Art does not appear, please install the **K-Lite Codec Pack.**

## Features

- High-quality audio download at `320 kbps`
- No Spotify account required
- Auto refresh token
